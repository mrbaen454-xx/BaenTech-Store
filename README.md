# BaenTech Store

BaenTech Store adalah aplikasi e-commerce toko elektronik berbasis microservices.

## Tech Stack

- Java Spring Boot
- Spring Cloud Eureka
- Spring Cloud Gateway
- Spring Security JWT
- PostgreSQL
- Maven

## Services

- discovery-service
- gateway-service
- auth-service
- product-service

## Current Features

### Auth Service
- Register
- Login
- JWT Authentication
- Role ADMIN dan USER
- Current user endpoint

### Product Service
- CRUD Category
- CRUD Product
- Search Product
- Filter Product
- Upload Product Image
- JWT Security
- ADMIN access for create, update, delete
- Public access for GET products/categories
